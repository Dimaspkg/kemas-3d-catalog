# Jenis File yang Didukung

Dokumen ini merangkum semua jenis file yang didukung untuk diunggah melalui panel admin aplikasi.

---

## 1. Model 3D Produk

File-file ini digunakan untuk menampilkan model interaktif pada kanvas 3D. Anda dapat mengunggah model untuk keadaan "tertutup" dan "terbuka".

-   **Ekstensi yang Didukung**:
    -   `.glb` (GL Transmission Format Binary - **Direkomendasikan**)
    -   `.gltf` (GL Transmission Format)

---

## 2. Gambar Produk

Gambar-gambar ini ditampilkan di galeri halaman detail produk.

-   **Ekstensi yang Didukung**:
    -   `.jpg` atau `.jpeg`
    -   `.png`
    -   `.webp`

---

## 3. Tekstur Material (PBR)

Tekstur digunakan dalam Manajemen Material untuk membuat bahan kustom. Anda dapat mengunggah peta tekstur yang berbeda untuk properti PBR (Physically-Based Rendering).

-   **Jenis Peta**: Base Color, Normal, Roughness, Metalness, Ambient Occlusion (AO).
-   **Ekstensi yang Didukung**: Semua format gambar web umum didukung.
    -   `.jpg` atau `.jpeg`
    -   `.png`
    -   `.webp`

---

## 4. Lingkungan 3D (Environment)

File-file ini digunakan sebagai latar belakang dan sumber pencahayaan berbasis gambar (IBL) di kanvas 3D.

-   **Ekstensi yang Didukung**:
    -   `.hdr` (High Dynamic Range)
    -   `.exr` (OpenEXR)

---

## 5. Logo Situs

Logo ini ditampilkan di header utama situs web.

-   **Ekstensi yang Didukung**:
    -   `.jpg` atau `.jpeg`
    -   `.png`
    -   `.webp`
    -   `.svg` (Scalable Vector Graphics)
